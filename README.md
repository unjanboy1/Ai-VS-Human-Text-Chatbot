







🤖 AI vs Human Text Chatbot

AI vs Human Text Chatbot is a machine learning-powered web application that analyzes text and predicts whether the content was written by a human or generated by artificial intelligence. The system uses Natural Language Processing (NLP) and Machine Learning techniques to identify patterns, writing styles, and linguistic characteristics that distinguish AI-generated content from human-written text.

📌 Features
🔍 Detects whether text is AI-generated or Human-written
⚡ Real-time text analysis and prediction
📊 Displays prediction confidence score
🤖 Interactive chatbot-style interface
🧠 NLP-based text preprocessing
📈 Machine Learning classification model
💻 User-friendly web interface
📱 Responsive design for different devices
🚀 Fast prediction results
